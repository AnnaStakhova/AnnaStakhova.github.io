Hi, I am Anna

I am a Graphic and Web Designer based in Cambridge UK.

I have professional experience in designing different types of printed materials. A couple of years ago, I started to learn web and discovered new interesting possibilities. I like to keep things simple and functional at the same time.

Graphic Design

I can imagine, create in editor and prepare to print a lot of advertising materials and make a simulation of 3d objects. I can design logos, key visuals and corporate style, mainly using Photoshop, Illustrator and a pencil.

Web Design

I am a web designer. I can create websites and know how to make it running across devices using the state-of-art technologies. I always keep learning and try to be up to date with the web design trends.
